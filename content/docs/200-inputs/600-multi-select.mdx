Add a _multi-select_ input to your app to allow users to select multiple values from a list of options.

## Options

Multi-select options can be added manually or pulled from a remote source.
The input ID in the URL template for your _multi-select_ input is replaced with the selected optionâ€™s values.

### Manual Options

Select the "manual" options type, then add options to your _multi-select_ input and set a default option.

### Remote Options

Select the "remote" options type, then specify the options URL and any required headers.

## Return Format

The array of options are JSON stringified and URI encoded. The values are available via query params.

**URL**
```
?myQueryParameter=%5B%22option-a%22%2C%22option-b%22%5D&myOtherQueryParameter=...
```

Make sure to parse the query parameter in your app code.

**App**
```TypeScript
const { myQueryParameter } = getQueryParams();

const multiSelectOptions = JSON.parse(myQueryParameter)
// [ 'option-a', 'option-b' ]
```

## Validation

The _multi-select_ input supports the following validation options:

- **Required:** Prevent your app from rendering until an input value is provided.
